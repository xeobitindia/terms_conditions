Privacy Policy

Last updated: 07-02-2026

This Privacy Policy explains how we collect, use, and protect user information when you use our application (the “App”).

1. Information We Access

Our App uses Google OAuth to access a user’s Gmail account only after explicit user consent.

The App may access:

  • Gmail message metadata and content
  • Email attachments limited to calendar invitation files (.ics)

We do not access unrelated emails or attachments beyond what is required for the App’s functionality.

2. How We Use Information

The App accesses Gmail data solely to:

  • Scan incoming emails for calendar invitation (.ics) files
  • Extract meeting details such as date, time, and event metadata
  • Present or process meeting invite information for the user

We do not use Gmail data for advertising, analytics, profiling, or resale.

3. Data Storage and Retention

  • Gmail data is processed in real time or stored temporarily as required
  • Email content is not permanently stored unless explicitly required
  • Any stored data is deleted when no longer needed

4. Data Sharing

We do not sell, share, or transfer user data to third parties.

5. Data Security

Reasonable technical and organizational measures are implemented to protect user data from unauthorized access or misuse.

6. User Control and Revocation

Users can revoke Gmail access at any time via their Google Account security settings. Once access is revoked, the App will no longer access Gmail data.

7. Google API Services Compliance

The App complies with the Google API Services User Data Policy, including the Limited Use requirements. Gmail data is used only to provide the user-facing features described above.

8. Changes to This Policy

This Privacy Policy may be updated periodically. Changes will be reflected on this page with an updated date.
